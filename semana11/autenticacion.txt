La autenticación o login permite identificar a un usuario antes de darle acceso a una aplicación. Gracias al inicio de sesión, cada usuario puede acceder únicamente a las funciones que le corresponden. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes dentro del sistema. Esto evita que personas no autorizadas accedan o modifiquen los datos. No se deben exponer públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni cualquier otro dato sensible de la comunidad.
