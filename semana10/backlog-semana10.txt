BACKLOG PRIORIZADO

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia.

3. Como miembro del equipo quiero consultar los reportes registrados para realizar seguimiento.

4. Como administrador quiero actualizar el estado de las incidencias para informar el progreso.

5. Como ciudadano quiero consultar el estado de mi reporte para conocer el avance de la gestión.
